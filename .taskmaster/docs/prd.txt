# BIF-AI: 경계선 지적 기능 성인을 위한 실시간 인지 지원 시스템

## 1. 프로젝트 개요

### 1.1 비전
"따뜻한 기술로 일상의 장벽을 허물어, BIF 성인들이 자립적이고 자신감 있게 살아갈 수 있도록 돕는다"

### 1.2 미션
경계선 지적 기능(IQ 70-85) 성인을 위한 실시간 상황 인식 기반 인지 지원 시스템을 개발하여, 일상생활에서의 어려움을 줄이고 독립적인 삶을 가능하게 한다.

### 1.3 핵심 가치
- **사용자 중심**: BIF 사용자의 인지적 특성을 고려한 직관적이고 단순한 인터페이스
- **프라이버시 우선**: 엣지 컴퓨팅을 활용한 개인정보 보호
- **실시간 지원**: 3초 이내 응답으로 즉각적인 도움 제공
- **포용적 기술**: 모든 사람이 기술의 혜택을 누릴 수 있도록

## 2. 타겟 사용자

### 2.1 주 사용자: BIF 성인
- **특징**: IQ 70-85, 일상생활 가능하나 복잡한 상황에서 도움 필요
- **니즈**: 
  - 복잡한 작업의 단계별 안내
  - 안전한 이동 경로 제공
  - 일정 및 약물 복용 관리
  - 긴급 상황 대응 지원

### 2.2 부 사용자: 보호자/돌봄제공자
- **특징**: BIF 성인을 돌보는 가족, 사회복지사, 의료진
- **니즈**:
  - 실시간 안전 모니터링
  - 활동 패턴 분석 리포트
  - 긴급 상황 알림
  - 원격 지원 기능

## 3. 핵심 기능

### 3.1 실시간 인지 지원
- **상황 인식**: 웨어러블 카메라를 통한 실시간 환경 분석
- **맥락 이해**: LLM 기반 상황 해석 및 적절한 행동 제안
- **단순 안내**: 5학년 수준의 쉬운 언어로 단계별 지시

### 3.2 안전 내비게이션
- **인지 부하 최적화**: 복잡한 경로 대신 단순하고 안전한 경로 제공
- **랜드마크 기반 안내**: 기억하기 쉬운 시각적 표지물 활용
- **대중교통 지원**: 버스/지하철 이용 시 단계별 안내

### 3.3 세이프티 넷
- **낙상 감지**: MediaPipe Pose를 활용한 실시간 자세 분석
- **위치 추적**: GPS 기반 안전 구역 이탈 알림
- **긴급 호출**: 자동/수동 긴급 상황 신고 기능

### 3.4 일정 관리 자동화
- **패턴 학습**: 사용자 행동 패턴 분석을 통한 일정 자동 생성
- **능동적 알림**: 약물 복용, 병원 방문 등 중요 일정 리마인더
- **활동 추적**: 일상 활동 기록 및 분석

## 4. 기술 스택

### 4.1 백엔드
- **프레임워크**: Spring Boot 3.5.0
- **언어**: Java 17
- **데이터베이스**: MySQL 8.0 (주 DB), Redis (캐싱)
- **인증**: Spring Security + JWT

### 4.2 AI/ML
- **LLM**: OpenAI API (상황 해석)
- **객체 인식**: YOLOv8
- **자세 추정**: MediaPipe Pose
- **OCR**: Google Vision API

### 4.3 인프라
- **클라우드**: AWS (EC2, RDS, S3)
- **컨테이너**: Docker
- **CI/CD**: GitHub Actions
- **모니터링**: CloudWatch, Prometheus

### 4.4 클라이언트
- **모바일 앱**: React Native (크로스 플랫폼)
- **웨어러블**: 목걸이형 카메라 (자체 개발)
- **통신**: WebSocket (실시간), REST API

## 5. 시스템 아키텍처

### 5.1 하이브리드 AI 아키텍처
- **엣지 컴퓨팅**: 민감한 데이터의 로컬 처리
- **클라우드 LLM**: 복잡한 상황 해석 및 의사결정
- **실시간 스트리밍**: WebRTC 기반 저지연 비디오 전송

### 5.2 마이크로서비스 구조
- **인증 서비스**: 사용자 인증 및 권한 관리
- **컨텍스트 분석 서비스**: 실시간 상황 인식
- **안내 생성 서비스**: BIF 최적화 안내문 생성
- **알림 서비스**: 푸시 알림 및 긴급 신고

## 7. 성공 지표

### 7.1 기술적 지표
- API 응답 시간 < 500ms
- AI 분석 시간 < 3초
- 시스템 가용성 > 99.5%
- 동시 사용자 100명 이상 지원

### 7.2 사용자 경험 지표
- 사용자 만족도 > 4.0/5.0
- 일일 활성 사용자 > 70%
- 긴급 상황 대응 성공률 > 95%
- 보호자 안심도 증가율 > 80%

### 7.3 사회적 영향 지표
- BIF 성인 자립 생활 기간 증가
- 보호자 부담 감소율
